From this preprint [\[1\]](https://figshare.com/articles/preprint/Closed_form_formulas_on_the_sums_of_three_cubes_for_k_114_192_/30509981?file=61812286), 

$\Biggl( -\alpha + \sqrt{(6n + \alpha)^2 + \frac{36n^3 - 19}{\alpha}} \Biggl)^{3} + \Biggl( -\alpha - \sqrt{(6n + \alpha)^2 + \frac{36n^3 - 19}{\alpha}} \Biggl)^{3} + \Biggl( 2 \cdot \alpha + 6n \Biggl)^{3} = 114$. 

To ensure that integer solutions are yielded, 

$y^2 = (6n + \alpha)^2 + \frac{36n^3 - 19}{\alpha}$
